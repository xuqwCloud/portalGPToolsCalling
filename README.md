# portalGPToolsCalling
Portal for ArcGIS内置的GP工具调用示例

#### 介绍

Portal for ArcGIS内置的GP工具调用示例，此示例使用portal内置的缓冲区分析GP工具。

#### 软件架构

此项目demo演示了如何在代码中调用Portal for ArcGIS内置的GP工具的过程，项目代码采用react脚手架搭建。

#### 代码配置说明

1.  项目代码全部在`App.js`这个文件中
2.  只需要更改代码文件中的portal地址、用户名/密码、测试服务地址即可

#### 安装和使用教程

1.  下载项目后解压；
2.  进入项目根目录，通过命令`npm i`安装相关依赖
3.  安装依赖之后通过命令`npm start`启动项目
4.  浏览器通过地址`localhost:3000`查看
